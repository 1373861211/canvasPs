# 批批拼拼

### 简介

#### 这是一个集成画笔、文字、橡皮檫、吸色器、撤销功能及拼图的图片处理工具，未来会兼容小程序、PC端

## 技术栈

1、vue3

2、ts

3、vite

4、canvas

上面全部都是初学技术栈，希望看到的大佬闲暇时，能给予反馈、指正	。

## 愿景

### 	在离职前，实现并优化全部功能，代码结构清晰、规范，高内聚低耦合，达到运行流畅，交互优美的效果，希望简历增色，技术能力得到一定程度的提高
